The Masterclass Agenda

A rigorous, 10-week modular syllabus designed to take students from core foundations to deploying production agentic systems.

1. Fundamentals

GenAI vs Traditional AI, LLMs, SLMs & Architecture.

2. RAG Mastery

Text embeddings, vector storage indexation, document chunking parameters, semantic search, and reranking.

3. AI Agents

ReAct structures, Planner/Executor patterns, function and tool calling configurations, and memory control loops.

4. Multi-Agent

Supervisor-worker patterns, agentic specialization, cooperative workflows, and communication orchestration models.

5. MCP Protocol

Model Context Protocol (MCP), building custom servers, securing local context data pipelines to remote LLMs.

6. Hands On

Weekend cohorts, 2-hour intensive live code labs, peer evaluations, and placement portfolio reviews.

Expand Syllabus Details

Weeks 1–2: Core Foundations +

Dive deep into LLMs vs SLMs, understanding context window architectures, token limitations, prompt designs, and local model serving configurations.

Week 3: Retrieval-Augmented Generation (RAG) +

Learn to ingest document datasets (PDFs, Markdown), configure document chunking sizes, build vector representations, select vector store endpoints, and execute semantic query routing.

Weeks 4–6: AI Agents & Logic Controllers +

Code logical loop workflows in Python. Configure LLMs to call custom external tools, parse parameters, manage short-term session state, and evaluate response logic loops.

Weeks 7–8: Multi-Agent Architectures +

Design a system of cooperative agents. Set up a supervisor agent that delegates coding or editing tasks to specialized sub-agents and reviews their results dynamically.

Weeks 9–10: MCP Protocol & Placement Capstone +

Build custom servers on the open-source Model Context Protocol (MCP) framework. Connect remote LLMs safely to local databases and files. Complete a showcase portfolio project ready for placement applications.
